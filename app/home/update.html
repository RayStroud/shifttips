Updates / Changes

Tue Oct 25, 2016

On the summary and period summary pages, there is an option to see either the averages or totals or both. Read more in the "How to use the summaries?" section on the About / Help / FAQ page.

Thu Oct 20, 2016

Finally updated the app with a lot of features that have been building up. If there are any troubles, please Email the Administrator. The old version is also online, if needed. The new features include:

  • • The summary page now has an option to view shift summaries grouped by length (in half-hours).
  • • The text-size option was added to the Filters bar. Read more in the "How to use the filters?" section on the About / Help / FAQ page.
  • • The view settings on the Account page are now collapsible for easier access on mobile devices. Users can save the current view settings, which can be reloaded on any device. Read more in the "How to use the view settings?" section on the About / Help / FAQ page.
  • • The filters and view settings are stored differently behind-the-scenes. Hopefully this doesn't cause any problems.
  • • The login process has changed behind-the-scenes, which will likely cause the app to prompt for a re-login.
